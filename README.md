# Project: Applications dedicated to Outlier or Novelty Detection for bicycle traffic metering systems in Nantes

The purpose of this project is to implement applications for automatic detection of outliers or event changes in bicycle count data for the city of Nantes. The goal is to determine whether cycling traffic datasets contain irregularities or significant variations linked to specific events.
The implemented approaches include:
- Novelty detection using an unsupervised model.
- Outlier detection using unsupervised models and a supervised model.

The machine learning models, from the PyOD library, are trained on the following dataset: [Nantes Metropolitan Area Bicycle Counts - Years 2014 to 2019](https://www.data.gouv.fr/datasets/comptages-velo-de-nantes-metropole-annees-2014-a-2019/)



# Projet : Applications dédiées à la détection des valeurs aberrantes ou des changements d’évènements pour les systèmes de comptage du trafic des vélos à Nantes

Ce projet vise à développer des applications capables de détecter automatiquement les valeurs aberrantes ou les changements d’évènements dans les données de comptage des vélos de la ville de Nantes. L’objectif est d’identifier si les jeux de données de trafic cyclable contiennent des irrégularités ou des variations significatives liées à des évènements particuliers.
Les approches mises en œuvre incluent :
- La détection des changements d’évènements avec un modèle non supervisé.
- La détection des valeurs aberrantes avec des modèles non supervisés et un modèle supervisé.

Les modèles de Machine Learning, issus de la librairie PyOD, sont entraînés sur le jeu de données suivant : [Comptages vélo de Nantes Métropole - Années 2014 à 2019](https://www.data.gouv.fr/datasets/comptages-velo-de-nantes-metropole-annees-2014-a-2019/)
